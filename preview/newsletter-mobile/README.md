# preview/newsletter-mobile/

이 폴더는 향후 독립 모바일 브라우저 프리뷰를 위한 자리이며, 현재는
HTML/CSS/JS를 구현하지 않는다. latest.html과 production 홈페이지에는
연결하지 않는다.

## 폴더 목적
Stitch 디자인이 승인된 이후, 실제 브라우저에서 열어볼 수 있는 독립 모바일
프리뷰(뉴스레터 내용을 브라우저 페이지로 재현한 것)를 구현할 자리다.
(`docs/newsletter/stitch-workflow.md`의 Gate 4)

## 입력 파일
- `../../design/newsletter/stitch/` 하위의 승인된 Stitch 자료
- `../../design/newsletter/contracts/` 계약 문서

## 출력 파일
없음(아직 구현 전).

## production 여부
아니오 — 이 폴더가 채워져도 `latest.html`/`index.html`과는 별개의 독립
페이지다.

## 사용자 승인 필요 여부
예 — Gate 2/3(Stitch 390/320 후보) 통과 후, Gate 4 시작 자체에 승인 필요.

## 금지 작업
- `latest.html`이나 홈페이지 `index.html`에서 이 폴더를 참조/링크하는 것
- Gate 4 승인 전에 HTML/CSS/JS를 작성하는 것
